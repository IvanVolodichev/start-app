Описание вайфреймов админ-панели

1. Общая структура админ-панели

Админ-панель предназначена для администраторов сайта и предоставляет следующие возможности:

  • Просмотр общей статистики по пользователям и событиям
  • Отслеживание активных, завершенных событий
  • Мониторинг обращений пользователей и жалоб
  • Просмотр журнала последних действий в системе

2. Десктопная версия

Десктопная версия админ-панели имеет следующие особенности:

2.1. Шапка сайта

  • Логотип слева
  • Горизонтальное меню навигации по центру
  • Профиль пользователя (администратора) справа

2.2. Статистические карточки

  • Расположены в два ряда по 4 карточки
  • Первый ряд: Пользователи, Активные события, Завершенные события, Всего событий
  • Второй ряд: Всего обращений, Необработанные обращения, Всего жалоб, Необработанные жалобы
  • Каждая карточка содержит иконку, название и числовое значение
  • Иконки имеют отличительные цвета для улучшения визуального восприятия

2.3. Таблица активности

  • Расположена под статистическими карточками
  • Содержит заголовок и таблицу с данными
  • Столбцы таблицы: Тип действия, Описание, Дата, Статус
  • Статусы имеют цветовую кодировку (зеленый - завершено, синий - активно, желтый - обрабатывается)

В десктопной версии все элементы располагаются горизонтально, используя преимущества широкого экрана, что позволяет увидеть больше информации без необходимости скроллинга.

3. Мобильная версия

Мобильная версия адаптирована для устройств с маленьким экраном и имеет следующие особенности:

3.1. Шапка сайта

  • Компактный логотип слева
  • Кнопка открытия мобильного меню (бургер) по центру
  • Аватар администратора справа
  • Основное меню скрыто и открывается по нажатию на кнопку

3.2. Навигационные вкладки

  • Добавлены горизонтальные вкладки для переключения между разделами
  • Предполагается, что на разных вкладках будут показаны разные наборы данных
  • Вкладки помогают экономить место на экране, разделяя информацию

3.3. Статистические карточки

  • Представлены в одну колонку, каждая карточка занимает всю ширину экрана
  • Уменьшены размеры иконок и текста
  • Сохранены цветовые коды для быстрой визуальной идентификации

3.4. Список активности

  • Таблица заменена на вертикальный список карточек
  • Каждая карточка содержит все данные о действии
  • Данные представлены в виде пар "ярлык-значение"
  • Статусы отображаются в виде цветных индикаторов внизу каждой карточки

В мобильной версии основной упор сделан на вертикальное расположение элементов, с фокусом на важнейшие данные. Таблицы трансформированы в списки для лучшей читаемости на маленьких экранах.

4. Адаптивные особенности

При переходе от десктопной к мобильной версии происходят следующие изменения:

  1. Изменение навигации с горизонтального меню на кнопку мобильного меню
  2. Перестроение статистических карточек из сетки 4x2 в одну вертикальную колонку
  3. Добавление системы вкладок для разделения информации
  4. Трансформация табличных данных в вертикальные списки
  5. Уменьшение размеров шрифтов, отступов и размеров элементов
  6. Скрытие некритичной информации или доступ к ней через дополнительные действия

Эти изменения обеспечивают оптимальное отображение администраторского интерфейса как на больших экранах настольных компьютеров, так и на мобильных устройствах, сохраняя при этом всю функциональность.
